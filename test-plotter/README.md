# Test Plotter

Simple Python program to plot MediaPipe points on your face. Clicking the points will print out all the selected in the order you clicked them. Useful for choosing the points the robot will draw.

## Usage

1. 
```
python3.12 -m venv .env
```
2.
```
source .env/bin/activate
```
3.
```
pip install -r requirements.txt
```
4.
```
python main.py
```
